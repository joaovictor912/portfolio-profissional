Flutter + Python + IA

Corretor de Provas

Detalhes do Projeto

Uma aplicacao movel multiplataforma, desenvolvida em Flutter, que automatiza a correcao de provas de multipla escolha. Utilizando a camera do dispositivo, o sistema aplica tecnicas de visao computacional + inteligencia artificial com um backend em Python para analisar as folhas de resposta, comparar com um gabarito mestre e fornecer o resultado em um curto espaco de tempo. O projeto inclui gestao de turmas e provas, armazenamento persistente de dados e a capacidade de exportar relatorios detalhados para ficheiros Excel (.xlsx).

Data Science + ML

Previsao Formula 1

Detalhes do Projeto

Este projeto de ciencia de dados desenvolve e avalia modelos de machine learning para simular o grid de largada de futuras corridas de Formula 1. O sistema utiliza dados historicos oficiais para treinar modelos preditivos e aplica fatores de desempenho calculados para refinar os resultados, gerando uma classificacao simulada que leva em conta a performance historica de equipes e pilotos.

NLP + Embeddings

Art Seeker

Detalhes do Projeto

O Art Seeker e uma aplicacao de Inteligencia Artificial voltada para a busca interativa em grandes acervos de obras de arte. A solucao utiliza modelos de NLP (Processamento de Linguagem Natural) para compreender consultas em linguagem natural. O sistema transforma essas consultas em vetores semanticos (embeddings) e realiza a comparacao com descricoes das obras no acervo.

Monte Carlo + Random Forest

Previsao Mundial

Detalhes do Projeto

Este projeto e um simulador que utiliza a Simulacao de Monte Carlo para estimar a probabilidade de cada time vencer o Mundial de Clubes 2025, executando o torneio 1000 vezes. A previsao de cada partida individual e realizada por um algoritmo de Machine Learning, o Random Forest, que calcula as chances de vitoria, empate ou derrota com base em estatisticas detalhadas.
